# 📄 AI CV Generator

A professional web application built with **React**, **TypeScript**, and **Lucide React** to create high-quality CVs/Resumes.

## 🚀 Features

- **Live Preview:** See your CV update in real-time as you type.
- **FILL IN THE BLANKS:** Pre-fill the form with your current profile data (based on your Faizal Malek profile).
- **Multiple Templates:**
  - **Professional:** The clean, minimal design used in `Faizal_Malek_CV_2026.pdf`.
  - **Modern:** A trendy gradient-based design with colorful badges.
- **Direct PDF Export:** High-quality PDF generation using `html2pdf.js`.
- **Responsive Layout:** Works on desktops and laptops for comfortable editing.

## 🛠️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run in development mode:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/templates/`: Contains HTML/CSS templates for different CV styles.
- `src/types/`: TypeScript interfaces for the CV data structure.
- `src/App.tsx`: Main application logic and form management.
- `src/App.css`: Premium UI styling for the generator.

## 📝 Usage

Simply enter your information into the categorized tabs (Personal Info, Summary, Experience, etc.). Click the **Generate PDF** button to download your CV instantly!
